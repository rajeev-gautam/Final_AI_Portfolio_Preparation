# **AI / Machine Learning Internship Portfolio**

This repository contains a comprehensive collection of industry-oriented AI, Machine Learning, Deep Learning, NLP, and Computer Vision projects developed during an intensive internship program.

All projects are implemented end-to-end with a strong focus on:

- clean code  
- real-world problem solving  
- proper documentation  
- reproducible experiments  

Most models were developed and trained using **Google Colab** for efficient experimentation and GPU support.

## 🚀 How to Run the Projects

1. Open the respective project folder  
2. Read the project-level `README.md`  
3. Open the notebook (`.ipynb`) in **Google Colab**  
4. Upload dataset or connect to Kaggle (if required)  
5. Run cells sequentially  

> **Note:** Large datasets are not uploaded to GitHub. Dataset links are provided inside project READMEs.
